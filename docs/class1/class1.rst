Class1 - Intro to NGINX Plus
============================

This class provides hands-on experience using NGINX plus.
==========================================================================

This lab is an adaptation of Agility lab "(Class-1_) - Intro to NGINX Plus" and has been modified by Chris Adrian and Bret Packard for SE Onboarding.

The class will cover the following topics:

- Install NGINX Plus and NGINX command line basics
- NGINX Plus HTTP Load Balancing

Expected time to complete: **30 minutes**

We will be using a jumpbox to connect to the lab environment. Got to the next
section for details on how to connect to the lab environment.
